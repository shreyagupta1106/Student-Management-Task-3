# SQL Data Analysis Internship – Task 3
Project Overview
This project focuses on analyzing student performance using advanced SQL concepts such as **JOINs, GROUP BY, HAVING, aggregate functions, and subqueries**.

##  Technologies Used

* MySQL
* SQL

##  Database Structure

* **Students** – Stores student information.
* **Courses** – Stores course details.
* **Enrollments** – Stores student-course enrollments and marks.

##  SQL Queries Implemented

* Top student in each course
* Pass rate per course (Marks ≥ 40)
* Overall topper across all courses
* Students enrolled in multiple courses

##  SQL Concepts Covered

* INNER JOIN
* GROUP BY
* HAVING
* Aggregate Functions (`COUNT`, `SUM`, `MAX`)
* Subqueries

##  Learning Outcomes

* Performed real-world SQL data analysis.
* Improved understanding of relational databases.
* Practiced advanced SQL queries used in data analytics.
